# Marketing Videos

Generated by Remotion compositions in `/remotion/`. Run from the repo root:

```bash
cd remotion && npm install && npm run render:marketing
```

This produces:

- `chravel-homepage-hero.mp4` — 1920×1080, ~18s, consumed by `src/components/landing/sections/HeroSection.tsx`.
- `chravel-homepage-hero-poster.jpg` — first-frame poster for `<video>`.
- `chravel-mobile-demo.mp4` — 1080×1920, ~13s, vertical mobile/app-store cut.
- `chravel-mobile-demo-poster.jpg` — poster for the mobile cut.

The hero component automatically falls back to the static `demo-preview-hero.webp`
image when the video file is missing, fails to load, or when the user has
`prefers-reduced-motion: reduce` enabled.

To preview compositions interactively: `cd remotion && npm run studio`.

To edit copy or scene timing, see:
- `remotion/src/compositions/HomepageHeroDemo.tsx`
- `remotion/src/compositions/MobileAppDemo.tsx`
- `remotion/src/components/SceneCaption.tsx`
