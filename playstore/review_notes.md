## Fast Validation Path (Android)
1. Launch app and sign in with demo credentials.
2. Open an existing trip (for example: "Bali Destination Wedding").
3. Validate core tabs:
   - Chat (send and receive)
   - Calendar (view itinerary)
   - Expenses (view splits)
   - Media (upload/access flow)
   - AI Concierge (prompt + response)
4. Open Settings → General Settings → Account Management → Delete Account (confirmation flow is fully implemented; no need to finalize deletion).
